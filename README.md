# 🔢 Pagination UI

Um projeto moderno de **Componente de Paginação**, desenvolvido para praticar conceitos de interfaces interativas utilizando **HTML5**, **CSS3** e estilização responsiva.

Ideal para websites, dashboards, blogs, sistemas administrativos e aplicações web que necessitam de navegação entre páginas de forma elegante e intuitiva.

---

## 📸 Preview do Projeto

Interface contendo:

- Navegação entre páginas
- Botões numéricos interativos
- Botões de avançar e voltar
- Destaque visual para página ativa
- Layout centralizado
- Design moderno e minimalista

---

## ✨ Tecnologias Utilizadas

- **HTML5**
- **CSS3**

---

## 🎨 Características do Design

- Interface clean e moderna
- Efeitos suaves de hover
- Destaque visual da página atual
- Bordas arredondadas elegantes
- Estrutura organizada e responsiva
- Paleta de cores moderna

---

## 🚀 Objetivo do Projeto

Praticar a criação de componentes reutilizáveis para interfaces web, explorando estilização avançada, alinhamento de elementos e experiência de navegação do usuário.

---

## 📚 Funcionalidades

- Navegação entre páginas
- Destaque para página ativa
- Botões de próxima e página anterior
- Estrutura reutilizável
- Interface responsiva
- Navegação intuitiva
